Welcome to sc-LIBRA documentation!
===================================

**sc-LIBRA** (/lu'make/) is a Python library for paired-Single-cell data integration and prediction
that uses DeepLearning Autoencoder model for it.
Find all code and data employed for developmental at  `LIBRA - Github <https://github.com/TranslationalBioinformaticsUnit/LIBRA>`_
where all *code* and *documentation* is providen for easy usage.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
